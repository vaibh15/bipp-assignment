# Data Consolidation and Transformation

This script is designed to process a specific Excel file named "Table_1_Annual_Series_For_Principal_Characteristics_2019_2020.xlsx." The script extracts relevant information from the Excel file, transforms the data, and then combines it into a single CSV file.

## Dependencies
pandas: The script utilizes the pandas library for data manipulation and analysis.
openpyxl: The openpyxl engine is used to read Excel files.

## Script Overview
- Table Cleanup
- Data Transformation
- Data Merging


## Note
Ensure that the Excel file and the script are in the same directory.
The script is tailored for the specific structure of the provided Excel file. Adjustments may be required for different formats.
Feel free to reach out for any clarifications or assistance!
